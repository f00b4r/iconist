# Iconist

Iconist is monorepo filled with many icon sets. Focus on minimal possible size of NPM package with SVG icons only. To use it on Vercel CDN `obr.now.sh`.

**Icon sets**

- dripicons
- emojitwo
- feathericons
- fontawesome
- heroicons
- ikonate
- lineawesome
- notoemoji
- openemoji
- remixicon
- twemoji

Thank you guys.
